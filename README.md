### Профилирование памяти
1. `pip install memory_profiler`
2. Добавляем перед основной функцией `@profiler`
```
@profile
def my_func():
    a = [1] * (10 ** 6)
    b = [2] * (2 * 10 ** 7)
    del b
    return a

if __name__ == '__main__':
    my_func()
```
3. Запускаем `python -m memory_profiler example.py`
